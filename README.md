# Simple File Manager

#### Only local non-root runs

### Functions
* two panes (Commander like)
* upload / download
* create
* simple text editor
* rename
* cut / copy / paste
* delete
* edit permissions
* trash
* dark theme
* adaptive design

### Platforms
* *nix only
* php 8.2 (min cli)
* php modules: zip

### Config
~$ nano src/config.php

### Run
~$ ./run.sh
