# Simple File Manager

<img width="1145" height="600" alt="image" src="https://github.com/user-attachments/assets/b44db3ff-bfc6-4aa6-b635-62691e3dfdb1" />

#### Only local non-root runs

### Functions
* two panes (Commander like)
* upload / download
* create
* simple text editor
* rename
* cut / copy / paste
* delete
* edit permissions
* trash
* dark theme
* adaptive design

### Platforms
* *nix only
* php 8.2 (min cli)
* php modules: zip

### Config
~$ nano src/config.php

### Run
~$ ./run.sh
