# Simple File Manager

<img width="1302" height="628" alt="image" src="https://github.com/user-attachments/assets/d9def366-55f6-4df0-bec9-e453b5292b0e" />


#### Only local non-root runs

### Functions
* two panes (Commander like)
* upload / download
* create
* simple text editor
* rename
* cut / copy / paste
* delete
* edit permissions
* trash
* dark theme
* adaptive design

### Platforms
* *nix only
* php 8.2 (min cli)
* php modules: zip

### Config
~$ nano src/config.php

### Run
~$ ./run.sh
