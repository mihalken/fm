# Simple File Manager

### Only local non-root runs

<img width="1302" height="628" alt="image" src="https://github.com/user-attachments/assets/d9def366-55f6-4df0-bec9-e453b5292b0e" />


### Functions
* two panes edge-to-edge
* upload / download
* create
* simple text editor
* rename
* file info
* cut / copy / paste
* delete
* edit permissions
* trash
* dark theme
* adaptive design

### Other functions
* media file info (ffmpeg ffprobe)

### Platforms
* *nix only
* php 8.2 (min cli)
* php modules: zip (download dir)

### Config
~$ nano src/config.php

### Run
~$ ./run.sh
